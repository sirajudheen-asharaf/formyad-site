# Privacy Policy **Effective Date:** April 2026 ForMyAd ("we", "our", "us") provides advertising analytics and reporting services. This Privacy Policy explains how we collect, use, and protect information. ## 1. Information We Collect We collect the following types of data: * Google Ads account data (campaign metrics such as impressions, clicks, conversions, cost) * Account identifiers (Customer ID, campaign ID) * User account information (name, email) We do NOT collect: * End-user personal data from ads (no PII) * Device-level tracking data ## 2. How We Use Data We use collected data strictly for: * Campaign performance reporting * Analytics and dashboard visualization * Internal optimization insights ## 3. Data Protection * All data is stored securely on protected servers * OAuth tokens are encrypted and never exposed to client-side applications * Access is restricted to authorized users only ## 4. Data Sharing We do NOT: * Sell data * Share data with third parties * Use data for advertising resale Data is used only within the ForMyAd platform. ## 5. Data Retention * Data is retained only as long as necessary for reporting purposes * Users can request deletion of their data at any time ## 6. Third-Party Services We use Google Ads API to retrieve campaign data. All usage complies with Google API policies. ## 7. User Rights Users can: * Request access to their data * Request deletion of their data * Disconnect their Google Ads account at any time ## 8.
